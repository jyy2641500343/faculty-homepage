连接 GitHub

使用仅限本仓库的 Fine-grained Token。令牌只保存在本次浏览器会话中，不会写入网站。

创建访问令牌
  1. 创建 Fine-grained personal access token，并将仓库访问范围限定为 faculty-homepage。
  2. 在 Repository permissions 中，将 Contents 设为 Read and write。
  3. 令牌建议设置较短有效期；编辑结束后可在 GitHub 中撤销。
