## 编辑与发布

1. 在管理页填写 GitHub 用户名、仓库名、分支和 Fine-grained personal access token。
2. Token 只需要当前仓库的 `Contents: Read and write` 权限。
3. 修改资料、新闻、论文或项目后，点击“保存并发布”。
4. GitHub Pages 通常会在 1–2 分钟内自动更新。

访问令牌仅保存在浏览器 `sessionStorage` 中，关闭浏览器标签页后会清除，不会写入仓库。
